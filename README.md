# my-dotfiles #

This repo includes my configuration files which can be managed by GNU Stow.

Repo structure

```
-dofiles
|-- git
|----|-- .gitmessage
|-- tmux
|----|-- .tmux.conf
```

## git ##

* .gitmessage

    prompt template for commit message. It will applied when committing.

## tmux ##

`.tmux.conf` is configurations of tmux.
